# Endpoints

## Supply to front-end

Supply to front-end, the route is prefixed by **/app**.

- Bus services
  - /app/services
- Bus stops
  - /app/stops
- Bus routes
  - /app/routes
- Bus arrival
  - /app/arrival

## From datamall

Getting from Datamall, the route is prefixed by **/datamall**.

- Bus services
  - /datamall/services
- Bus stops
  - /datamall/stops
- Bus routes
  - /datamall/routes
